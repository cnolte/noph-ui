# Noph UI

Noph UI is a component library for Svelte following Material Design 3. Its goal is to reduce the amount of JavaScript by replacing it with CSS and HTML.

## Installation

```bash
npm install noph-ui
```

## Components

- Button (in progress)
- Divider (in progress)
- List (in progress)
- Menu (in progress)
- Progress (in progress)
- Radio (waiting for start)
- Ripple (in progress)
- Segmented Button (in progress)
- Snackbar (in progress)
- Switch (witing for start)
- Tooltip (in progress)
- Textfield (waiting for start)
