# Noph UI

Noph UI is a component library for Svelte following Material Design 3. Its goal is to reduce the amount of JavaScript by replacing it with CSS and HTML.

## Installation

```bash
npm install noph-ui
```

## Components

Beta (No breaking changes expected)

- Buttons
- Icon buttons
- Progress indicators
- Ripple

In progress (Breaking changes expected)

- Lists
- Menus
- Segmented buttons
- Snackbar
- Tooltips

Pending

- Chips
- Dialogs
- Radio
- Select
- Sliders
- Switch
- Tabs
- Text fields
